Import Products using Excel

Import your company products into Odoo using Excel.

Make sure you have xlwt Python libarary installed.

Following fields are import.

  • NAME
  • INTERNAL REFERENCE
  • BARCODE
  • UOM
  • PURCHASE UOM
  • SALE PRICE
  • COST PRICE
  • WEIGHT
  • VOLUME
  • SALE OK
  • DESCRIPTION
